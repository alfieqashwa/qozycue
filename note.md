# Video 1:

## Introduction

Saya ingin jelasin tentang apa itu Qozy Cue, apa motivasi pembuatan aplikasi ini.

Qozy Cue merupakan aplikasi online yg diciptakan untuk membantu para pelaku usaha billiard dalam memantau segala aktivitas usahanya serta mengelola transaksi yg terjadi, SECARA REAL TIME.

Sebatas yg saya ketahui, sistem yg digunakan di industri billiard masih didominasi oleh sistem berbasis offline, dimana keterbatasannya menjadikan para pemilik usaha untuk datang memantau aktifitas bisnis mereka, atau meminta staff mereka untuk mengirimkan data transaksi, misal, via email.

Di masa kini teknologi tsb bisa dikatakan usang, bila dibandingkan dengan teknologi di bidang lain, dimana sebagian besar telah beralih ke teknologi berbasis online dan tidak lagi menggunakan teknologi yg masih berbasis offline.

Awalnya, Qozy Cue hanya sekedar aplikasi online dan tidak real time, namun seiring berjalannya waktu saya menyadari bahwa akan mustahil untutk mampu mengakomodir kebutuhan multi-cashier dalam mengoperasikan transaksi dalam waktu bersamaan.

Akhirnya saya memutuskan untuk membuat ulang aplikasi yg tidak hanya sekedar berbasis online saja, tetapi juga real time, sehingga dapat mengakomodir pada skenario paling maksimal, yakni fitur multi-cashier dimana masing-masing dapat mengelola hingga di atas seratus meja dalam waktu bersamaan.

Di sisi lain, fitur real time inilah yang sebenarnya paling dibutuhkan oleh pihak owner atau para pemilik usaha bisnis billiard. Apalagi dalam satu usaha biasanya dimiliki lebih dari satu pemilik, maka para owner dapat memantau segala aktivitas serta transaksi di masing-masing device mereka secara real time.

Qozy Cue sangat responsive dan dapat digunakan di multi-platform; baik itu smartphone, tablet, PC, dan laptop.

Selanjutnya saya akan tunjukkan cara menggunakan Aplikasi Qozy Cue, di mulai dari bagaimana cara melakukan registrasi hingga mengelola transaksi.
